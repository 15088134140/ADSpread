# 新建发布计划弹窗 - 播放周期功能设计说明

## 需求变更概述
在原有新建发布计划弹窗基础上，增加「播放周期」选择功能：
- 候选项：周一至周日共7个选项
- 交互方式：支持多选（可同时选多天）
- 位置：放在生效结束时间之后

## 设计方案

### 1. 布局设计
- 采用网格布局展示周一至周日，一行7个按钮
- 整体区域带边框，与其他表单项保持一致的视觉风格
- 下方增加快捷操作区域，用分割线与选项区隔开

### 2. 交互设计
- 使用Checkbox实现多选功能，视觉上呈现为可点击的按钮样式
- 选中状态使用主色调填充，文字变为白色
- 悬停时有轻微变色反馈
- 提供快捷操作按钮：
  - **全选**：选择一周七天
  - **工作日**：只选择周一至周五
  - **周末**：只选择周六和周日
  - **清空**：取消所有选择
- 默认值：默认全选所有日期，符合大多数使用场景

### 3. 视觉设计
- 保持与Ant Design风格一致（根据需求文档参考截图推测）
- 色彩使用系统原有设计变量：
  - 主色：`#1890ff`
  - 边框色：`#d9d9d9`
  - 文字：主色`#262626`，辅助色`#8c8c8c`
- 圆角、间距、阴影都保持统一

### 4. 响应式适配
- 大屏幕：一行显示7天
- 小屏幕（手机）：自动变为一行4天，保证触摸操作舒适度
- 快捷操作按钮支持自动换行

## 实现要点
1. **数据存储**：提交时，选中的星期以数组形式提交（周一到周日对应数字 1-5,6,0）
2. **后端处理**：客户端拉取时，服务端根据当前日期的星期判断是否在播放周期内
3. **默认行为**：默认全选所有日期，对用户最友好，减少操作步骤
4. **可访问性**：使用原生Checkbox，支持键盘操作和屏幕阅读器
